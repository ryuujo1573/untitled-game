# Active Plans

Store in-progress execution plans here.

## Naming Convention

Use: YYYY-MM-DD-short-slug.md

Example: 2026-04-20-render-graph-cleanup.md

## Required Sections

- Context and goals
- Task breakdown
- Validation plan
- Progress log
- Decision log
- Risks and rollback notes
